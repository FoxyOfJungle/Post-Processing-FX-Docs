
## Effects Parameters List <!-- {docsify-ignore} -->

Use them to indicate which effect variable to modify. They start with a capital "PP_", to facilitate code-completion (intellisense).

Example:
```gml
ppfx_effect_set_parameter(ppfx_id, PP_EFFECT.ZOOM, PP_ZOOM_AMOUNT, 1.4);
```
In the example above, I am modifying the zoom amount to 1.4, using PP_ZOOM_AMOUNT.
